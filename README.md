# Финансовый Менеджер 2026

## Описание проекта
Простой и функциональный десктопный менеджер личных финансов, разработанный на Python с использованием Tkinter и Pandas. Проект предназначен для учета доходов и расходов конкретно на 2026 год, с возможностью анализа данных и построения графиков.

## Используемые технологии
*   **Python** (версия 3.14)
*   **Tkinter** (встроенный GUI)
*   **Pandas** (анализ и хранение данных в CSV)
*   **Matplotlib** и **Seaborn** (визуализация графиков)
*   **Unittest** (модульное тестирование)

## Архитектура проекта
Проект разделен на логические модули:
*   `main.py`: Точка входа, запуск тестов и GUI.
*   `models.py`: Описание класса `FinancialOperation`.
*   `storage.py`: Функции для загрузки, сохранения и фильтрации данных (работа с `data/finance_data_2026.csv`).
*   `gui.py`: Графический интерфейс пользователя (Tkinter).
*   `analysis.py`: Функции построения графиков (круговая диаграмма, динамика).
*   `utils.py`: Вспомогательные функции валидации данных.
*   `test_finance.py`: Unit-тесты для проверки логики.
*   `data/`: Папка для хранения файла базы данных.

## Установка и запуск
1. Клонируйте репозиторий или скачайте файлы проекта.
2. Установите необходимые библиотеки с помощью pip:
    ```bash
    pip install pandas matplotlib seaborn
    ```
3. Запустите основное приложение:
    ```bash
    python main.py
    ```

## Функционал
*   **Учет:** Добавление, редактирование и удаление доходов/расходов.
*   **Хранение:** Все данные сохраняются в файл `data/finance_data_2026.csv`.
*   **Валидация:** Строгая проверка дат (только 2026 год) и сумм.
*   **Анализ:** Автоматический расчет баланса, фильтрация по категориям, просмотр ТОП-расходов и динамики.

## Тестирование
При запуске `main.py` автоматически выполняются все модульные тесты (`test_finance.py`). Также тесты можно запустить через PyCharm.
